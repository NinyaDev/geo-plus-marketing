# Echo Skill

## Purpose
Development/testing skill that echoes back user input. Used to verify the agent pipeline is working end-to-end.

## Triggers
- `echo <message>` — echoes back `<message>`

## Examples
- Input: `echo hello world` → Output: `Echo: hello world`
- Input: `echo test 123` → Output: `Echo: test 123`

## Notes
- This is an MVP skill for pipeline verification
- Will be replaced by production skills in future iterations
